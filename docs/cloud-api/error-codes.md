# 任务错误码列表

本文档列出了SuperDock系列产品在执行各种任务时可能遇到的错误码及其说明。错误码按照功能模块进行分类，便于快速定位和解决问题。

## 错误码分类规则

- **60XXXX**：MSDK执行操作的错误码
- **61XXXX**：机场任务执行流程的错误码
- **62XXXX**：机场控制、任务前检测、设备执行和运行期的错误码
- **64XXXX**：执行飞行或加载航线的错误码
- **其他**：兼容大疆上云的错误码

## 错误码使用说明

### 错误处理建议

1. **60XXXX系列错误**：主要涉及MSDK操作，建议检查飞行器状态、网络连接和参数设置
2. **62XXXX系列错误**：机场控制、充电/换电、任务前检测、设备执行和运行期问题，建议检查机场硬件状态、电源供应、网络、RTK 和天气条件，如果无法自行解决联系售后团队.
3. **64XXXX系列错误**：该错误码是一般是遥控器MSDK加载航线和执行起飞任务出现问题的错误码.
4. **其他错误码**：兼容大疆标准，按照大疆官方文档进行处理
