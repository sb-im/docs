## 2026-04-24

### Added

- Added error codes for task workflow, Dock control, charging/battery swap, pre-task checks, device execution, AI detection, PSDK communication, and more.
- Added the L3 device type.

### Improved

- Improved the mobile menu bar layout to enhance navigation usability on phones.

### Fixed

- Corrected certain camera device types:
  - `P1` type corrected to `50`

### Newly Supported Devices

#### SuperDock Series Docks

- **SuperDock S25M4** - M4-25
- **SuperDock S25M400** - M400 charging
- **SuperDock S25M400S** - M400S battery swap

#### Compatible Drones

- DJI Matrice 400 Series

## v1.0.0-beta (Initial Beta Release)

### Release Date

2025-09-23

### Release Notes

This is the initial beta release of the SuperDock API documentation, providing basic developer documentation and an API reference.

### Key Features

#### Documentation Updates

- A brief introduction to SuperDock Docks based on the Cloud API protocol and relevant differences
- A brief introduction to the backend DEMO that adapts SuperDock Docks to the Cloud API
- FAQs about Dock operation

### Supported Devices

#### SuperDock Series Docks

- **SuperDock S22M300** - 2022 M300/M350 Dock version
- **SuperDock S2201** - 2022 Dock version
- **SuperDock S2301** - 2023 Dock version
- **SuperDock S24M350** - M350 Dock - 2024 version
- **SuperDock S24M350S** - M350 battery-swap Dock - 2024 version
