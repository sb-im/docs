# 概述

草莓创新SuperDock系列产品完全兼容DJI上云API标准，为客户提供无缝的云端集成体验。

## 为什么使用DJI上云API

### 降低迁移成本：

兼容该接口，客户可直接复用现有系统与经验，无需重新学习，大幅减少开发和培训成本。

### 提升系统兼容性：

通过标准化接口，SuperDock可与DJI设备统一管理，支持混合部署，简化集成流程，保证系统稳定性与灵活性。
